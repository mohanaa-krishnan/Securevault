# SecureVault

SecureVault is a password security project built in multiple phases.

## Goal

Build a secure password manager with:

- Password auditing
- Password generation
- Credential management
- Encryption
- Secure vault storage

## Tech Stack

- HTML
- CSS
- JavaScript
- Java
- Spring Boot
- MySQL

## Current Progress

- [x] Project initialized
- [x] Basic HTML form
- [x] Initial Java password validation
- [ ] Password scoring
- [ ] Frontend improvements
- [ ] Backend API
- [ ] Database integration
- [ ] Encryption

## Safety Note

This project is currently for learning only.

Do not use real passwords while testing early versions.
